HTML as One of My Core Skills

What HTML Is

HTML stands for HyperText Markup Language. It is a markup language used to structure content on web pages.

HTML is not used to program behavior. Instead, it is used to organize content such as headings, paragraphs, lists, links, images, tables, and forms so that browsers can display them properly.

Structural Elements

html is the root element of the whole HTML document. It contains all other elements.

head contains metadata such as the page title and character settings. It is used for information about the document, not visible page content.

body contains all the visible content of the webpage.

header is used for introductory content such as titles and opening information.

nav is used for navigation links.

main is used for the main content of the page.

section is used to group related content under one topic.

article is used for self-contained content that can stand on its own.

footer is used for ending content such as copyright or contact information.

Text Formatting Elements

h1 to h6 are heading elements used to organize content by importance.

p is used for paragraphs of text.

strong is used to show strong importance.

em is used to emphasize text.

br creates a line break within text.

hr creates a thematic break between topics.

Lists

ul creates an unordered list when order does not matter.

ol creates an ordered list when sequence matters.

li defines each item inside a list.

Links

a is the anchor element. It is used to create hyperlinks to pages, sections, email addresses, or external websites. It should always use the href attribute correctly.

Images

img is used to display an image. It should include the src attribute for the file location and the alt attribute for image description.

Tables

table is used to organize data into rows and columns.

tr creates a table row.

th creates a table heading cell.

td creates a table data cell.

caption gives a title to the table.

Forms

form is used to collect user input.

label describes a form field.

input creates different input fields.

textarea is used for longer text input.

button creates a clickable button.

select creates a drop-down menu.

option defines an option inside a drop-down menu.

Semantic Elements

Semantic elements clearly describe the purpose of content. Examples include header, nav, main, section, article, and footer.

They should be used to improve structure, readability, accessibility, and meaning.
